Python control for Broadlink RM2 IR controllers
===============================================

* Обработки зависимостей для broadlink==0.5
* Скачайте 
* `sudo git clone https://github.com/zver5/python-broadlink`
* перейдите в папку
* `cd python-broadlink`
* Запустите установку
* `sudo python setup.py install`
* Пользуйтесь как обычно
